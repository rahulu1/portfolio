Lua Scripting

Using Sol as its binding API, blitzENGINE utilizes a component-based architecture, where users can write their own custom game logic and components in Lua.

Make use of various lifecycle functions, callbacks, a built-in EventBus, and more to create dynamic, powerful components and actors. Developers can also utilize of a growing number of natively-integrated C++ components, provided to make development quicker, iteration easier, and games more performant.

SDL Integration

blitzENGINE is built on top of SDL, a cross-platform development library designed to abstract away platform-specific calls for input polling, graphics rendering, audio output, and more.

Through Sol, blitzENGINE's API provides a variety of flexible and powerful ways for developers to directly utilize SDL and its features for themselves, straight through Lua, with the added layer of cohesion and convenience you've come to expect from a game engine.

Box2D Physics

Watch your games spring to life as soon as you add a Rigidbody component into your scenes. Powered by Box2D, blitzENGINE's Physics API and Rigidbody component provide developers with a way to add engaging, performant physics simulations to their games. Create static, dynamic, and kinematic physics bodies with triggers and colliders to create complex interactions between your game's Actors.

bliztENGINE implements a fixed-timestep loop in order to ensure that physics simulations are decoupled from the game's performance on any specific hardware. What does that mean? Whether you're pushing 15 FPS or 150 FPS, physics simulations will play out the same, delivering exceptional accuracy for your games. blitzENGINE utilizes platform-specific code to ensure time is measured as precisely as possible, allowing for the fixed-timestep loop to perform as consistently and accurately as possible across a variety of platforms.

Extensive Tweening Support

blitzENGINE's crowning jewel and most unique feature, make use the engine's wildly expressive tweening API (based off of the incredible DOTween plugin for Unity) to take your scripting to the next level. All datatypes with support for basic arithmetic (and some without!) can have their values tweened using a host of different easing functions, loop types, callbacks, and more. Use the wildly powerful GOTween.To function to tween almost any type of value, or use the Tween shortcut functions present on Native blitzENGINE components to write even more concise, readable, and performant code.

Use callbacks, tween chaining, and more to quickly spin up incredibly polished and reactive game elements with very short iteration times. The tweening API is intuitive and expansive enough to script nearly anything you can think with 20% of the code and effort. Let GOTween do the heavy lifting, so you can focus on the creative side of game development and make your games the most unique experiences they can be!
